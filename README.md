# Slime EA — ORCA 슬라임 동적 배분 MT5 Expert Advisor

ORCA 전략(돌파+피라미딩+트레일링) 기반 MQL5 EA.
Physarum(점균) 알고리즘으로 심볼별 자본을 동적 배분.

## 파일

| 파일 | 설명 |
|------|------|
| `ORCA_FX.mq5` | 금(XAUUSD) 전용 — 진화 검증 완료 챔피언 DNA |
| `ORCA_Multi.mq5` | 멀티심볼(금,은,유달,나스닥,오일) 슬라임 배분 |
| `ORCA_OIL.mq5` | 오일(USOIL) 전용 — ORCA_FX 기반 파라미터 조정 |

## 전략

1. 241봉(20시간) 구간 고점/저점 돌파 감지
2. 방향 맞으면 피라미딩 최대 4단
3. 트레일링 0.1% + ATR×19.7 익절
4. 금요일 강제 청산 (주말 갭 방지)
5. 청산 후 재진입 차단 (고저점 갱신 시만 허용)

## 슬라임 배분 (ORCA_Multi)

샤프비율 기반 D값 EMA로 심볼별 자본 배분 동적 조절.
성과 좋은 심볼에 더 많은 자본 배분.
